# KT3-VUE

Интернет-магазин продуктов на Vue 3 с использованием Vuex для управления состоянием.

## Описание проекта

Данный проект представляет собой интернет-магазин продуктов с возможностью:
- Просмотра каталога товаров
- Отображения цен и скидок
- Управления состоянием товаров через Vuex
- Удаления товаров из каталога
- Адаптивного интерфейса

## Технологии

- **Vue 3** - фреймворк для создания пользовательского интерфейса
- **Vuex** - управление состоянием приложения
- **Vite** - инструмент для сборки проекта
